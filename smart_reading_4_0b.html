丑　石

扩展阅读 · 第四单元开篇 · 根据贾平凹《丑石》改写

我常常遗憾我家门前的那块丑石：它黑黝黝地躺在那里，跟一头牛似的；谁也不知道它是什么时候来到这里的，谁也不去理会它。只有到了麦收季节，门前摊了麦子，奶奶总是说："这块丑石，多碍手碍脚啊，什么时候把它搬走吧。"

🌏 Reading Note — 为什么叫“丑石”？
🪨
丑石 不是单纯说石头“难看”，而是说它在村里人的眼光里“没有用、不规则、碍事”。文章先让大家都看不起它，后面再慢慢反转。
📚
开头的情感词 遗憾 很重要：作者小时候也觉得这块石头不好，后面才明白自己以前看错了。

于是，伯父家盖房，想用它垒墙，但它极不规则，没棱角儿，也没平面儿；把它砸开吧，又懒得花那么大的力气，因为河滩并不很远，随便去找一块回来，哪一块也比它强。房子盖好了，垒台阶时，伯父也没有看上它。有一年，来了一个石匠，为我家洗一台石磨，奶奶又说："用这块丑石吧，省得从远处搬动。"石匠看了看，摇着头，说它不够粗糙，也没有用处。

📝 Vocabulary Note — 碍手碍脚 / 省得 / 规则
🔎
碍手碍脚 = in the way. 很口语，也很形象，表示一个东西放在那里妨碍做事。
🔎
省得… = so as to avoid trouble. 例如：早点出门，省得迟到。
🔎
规则 在这里指“形状规整”。文中说丑石“不规则”，就是它不好用作建筑材料。

它不像玉那样的细腻，可以在上面雕花刻字，也不像大青石那样的光滑，可以用来洗衣服；它静静地躺在那里，门前的树荫没有遮挡它，花儿也不在它身上生长。周围只有荒草，慢慢地，它居然长出了绿苔、黑斑。我们这些孩子，也讨厌起它来，曾经一起要搬走它，但力气不够；虽然大家骂它、嫌弃它，但也无可奈何，只好随它留在那里了。

🌏 Reading Note — 前半部分的作用
🧭
前半部分反复写丑石“没用”：不能垒墙，不能做台阶，也做不了石磨。这样的重复是为了给后面的“陨石”发现做铺垫。
🌿
作者还写绿苔、黑斑、荒草，这些细节让丑石显得更加不起眼，也让后面的反转更强。

稍稍能安慰我们的，是在那石上有一个不大不小的凹坑，雨天就盛满了水。常常雨过三天了，地上已经干了，那凹坑里还有水，小鸡就去那里喝水。每当到了十五的晚上，我们盼满月出来，就爬到丑石上，望着天边，奶奶总是要骂的，害怕我们摔下来。果然有一次我摔了下来，摔破了膝盖。

📝 Vocabulary Note — 果然 / 盼 / 稍稍
🔎
果然 = as expected. 前面奶奶总担心孩子摔下来，后面“果然”就真的摔了。
🔎
盼 比 “等” 更有期待感，表示心里很希望某件事快一点发生。

人人都骂它是丑石，它真是丑得不能再丑的丑石了。

终于有一天，村子里来了一个天文学家。他从我家门前路过，突然发现了这块石头，眼睛马上就直了。他再也没有离开，在这儿住了下来；以后又来了好多人，说这是一块陨石，从天上落下来已经有二三百年了，是一件了不起的东西。不久便来了车，小心翼翼地把它搬走了。

🌏 Cultural Note — 陨石与“反转”
✨
文章的最大转折在这里：原来人人看不起的石头，其实是从天上落下来的 陨石。这让“价值”不再由日常用途决定。
🧠
作者借这个反转在说：有些事物表面普通、甚至难看，却可能有完全不同的意义。不要只用“有没有立刻的用处”来判断它。

这使我们都很惊奇！这又怪又丑的石头，原来是天上的呢！它在天上闪过光，我们的祖先也许仰望过它，它给了他们光明、憧憬；而它后来落下来了，在脏土里、荒草里，一躺就是几百年了？！

📝 Vocabulary Note — 以丑为美 / 世俗 / 讥讽
🔎
以丑为美 = to treat ugliness as beauty. 这里不是说真的“丑就是好看”，而是说真正不凡的东西，往往不符合一般人的审美和标准。
🔎
世俗 指 ordinary worldly standards；讥讽 指 mocking ridicule。放在一起，就是“被一般人的偏见和嘲笑所误解”。

奶奶说："真看不出来！它那么不一般，却怎么连墙也垒不了，台阶也垒不成呢？"

"它是太丑了。"天文学家说。

"真的，是太丑了。"

"可这正是它的美！"天文学家说，"它是以丑为美的。"

"以丑为美？"

"是的，丑到极点，便是美到极点。正因为它不是一般的石头，当然不能去做墙、做台阶，不能去雕刻、洗衣服。它不是做这些小事情的，所以就常常受到世俗的讥讽。"

奶奶脸红了，我也脸红了。

我感到了自己的无知，也感到了丑石的伟大；我甚至怨恨它这么多年来默默地忍受着这一切，而我又立刻深深地感到它那种不怕被误解、寂寞地生存的伟大。
